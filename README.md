# mouse-gesture
一个轻量、纯本地、无隐私收集的浏览器鼠标手势扩展。 通过在页面中绘制鼠标轨迹，快速执行常用浏览器操作，提高日常浏览效率。

✨ 功能特性

🖱️ 鼠标手势操作

按住鼠标按键并拖动即可绘制手势

支持常见操作：

前进 / 后退

刷新页面

关闭当前标签页

恢复最近关闭的标签页

🎨 可视化手势轨迹

手势轨迹以 紫色加粗线条实时显示

清晰直观，不影响网页本身的交互

使用透明 canvas 叠加层绘制，不修改页面内容

⚡ 响应迅速、逻辑简单

手势识别逻辑清晰

本地实时处理，无延迟

🔒 注重隐私

不联网、不上传任何数据

不记录网页内容或浏览历史

所有逻辑仅在本地浏览器中运行

📦 安装方式
方式一：开发者模式安装（推荐用于开发/测试）

克隆或下载本项目：

https://github.com/dajiangfu/mouse-gesture.git


打开浏览器扩展管理页面

Chrome / Edge：chrome://extensions

开启右上角 开发者模式

点击 加载已解压的扩展程序

选择本项目根目录

🧭 使用说明

确保扩展已启用

在任意网页中：

按住指定鼠标按键（如右键）

拖动鼠标绘制手势轨迹

松开鼠标后：

若手势匹配预设规则，将执行对应操作

手势未识别时，不会触发任何行为

🧠 支持的手势示例
手势方向	功能
←	后退
→	前进
↑	刷新页面
↓	关闭当前标签页
其他	可按代码配置

（具体手势映射以代码或配置为准）

🔐 权限说明（简要）

访问网页（Host permissions）
用于监听鼠标轨迹并绘制手势线条

tabs
用于执行前进、后退、关闭标签页等浏览器操作

sessions
用于通过手势恢复最近关闭的标签页

storage
用于在本地保存扩展设置

所有权限仅用于实现鼠标手势这一单一用途。
